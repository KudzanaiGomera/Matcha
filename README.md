# Matcha

  Matcha is a mobile-friendly dating website, with some popular features from apps like Tinder as well as common features that are on most dating websites.

# Installation

  Go to : 
  
  matcha/
  >install python3,
  >pip3,flask
  >python3.run.py
  
  Enjoy!
